# Udupi Homestays

Rebuilt marketing site for Udupi Homestays, inspired by premium hospitality layouts with a green secondary palette.

## Getting started

- Install dependencies: `npm install`
- Run the dev server: `npm run dev`
- Production build: `npm run build`

## Notes

- Replace the placeholder text, imagery blocks, and contact details with your final content.
- Main pages live under `src/app` and shared UI sits in `src/components`.
